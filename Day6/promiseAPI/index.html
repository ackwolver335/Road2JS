Promise API and Methods

  • Promise.all() : for checking all Promises at once
  • Promise.allSettled() : for removing clearing error for single Promise
  • Promise.race() : Used for getting low-time Promise
  • Promise.any() : Used for getting low-time Promise without any error
  • Promise.resolve() : Returns the Promise and provides its resolved value
  • Promise.reject() : Returns the rejected error of the Promise with the value
